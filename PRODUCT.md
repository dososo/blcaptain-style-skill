# BLCaptain Style Skill — Product

> 一句话：**把一段中文内容，变成第一眼惊艳、一眼识别是「我们」的小红书 / 公众号图文卡片组。**
> 详细资产全貌见 [`docs/ASSET-OVERVIEW.md`](docs/ASSET-OVERVIEW.md)。

## 是什么 / 给谁

一个面向中文内容创作者的图文卡片生成 Skill（Claude Code / Codex）。给：写小红书 / 公众号 / X 的人，咖啡馆主理人、数码测评、活动策划、读书博主、做 AI 工具教程的人——任何需要「内容 → 专业图文」的人。

工作形态是**智能驱动**：Claude 读懂内容、亲手写结构化 `brief.json`，交给确定性引擎渲染成 PNG。内容理解归 Claude，确定性渲染 + 质量校验归代码——**不让 AI 自由画版面**（AI 在自由版面设计上平庸，给它被验证过的骨架，任务从「设计」降级成「填充」，成品稳定性立刻上来）。

## 解决什么问题

市面 AI 图文卡一眼假——Tailwind 大色块 + emoji 堆砌 + 紫蓝橙粉渐变 + Inter 字体，**「做的是网页，不是杂志」**。还有：配图难（要么自己传累、要么 emoji 假、要么一眼 AI 插画 + 水印被平台限流）、风格不搭平台（PPT 截成竖图节奏全错）、开发/demo 信息漏进用户卡面。

我们把这些变成**可执行的约束**——字号阶、留白比例、对比度阈值、断行规则、色板角色、独家影像配方，**落到代码里全是常量**。这些常量才是真正的护城河。

## 三套视觉语言（护城河）

每套有一句话灵魂主线，不是换色皮肤：

- **静纸 · Still Paper** — 把内容沉淀成有温度的纸本手记。暖纸 + 宋体 + 朱砂 + 大留白。7 封面母体 + 14 正文机制。
- **实证 · Signal Proof** — 界面是证据层，读者才相信流程。奶白档案纸 + 电蓝 + VERIFIED 印章。护城河是**证据方法论叙事**，不是漂亮数据卡。3 封面形态 + 7 证据卡。
- **图桥 · Bridge Canvas** — 把内容连接成跨平台统一表达。cinematic 满铺 + 电影黑边。5 版式 + 多平台同源（3:4 / 1:1 / 16:9）。

**独家影像配方**：静纸暖墨 duotone / 实证电蓝三调 + riso 半调 / 图桥 teal-gold split-tone + haze 散焦氛围底——同一张真实图变成「只有我们能出的影像」，一眼可辨、抄不走、不撞通用 stock。

## 能力边界（不接什么 = 产品定义）

明确不接：追星粉丝向 · 纯促销硬广 · >12 屏长教程 · 需大量专有 IP 图。碰到这些，开头就告诉用户「你可能想用别的工具」——**一个什么都能做的 Skill 最后通常什么都做不好。**

## vs 上游优秀项目（领先在哪，不复制）

研究借鉴业界优秀项目的产品化流程 / 固定版式思路 / 渲染链路 / 验证意识，但**不复制其设计、代码、模板、CSS、资产、配色、布局 ID 或视觉身份**。领先点：① **3 套视觉语言**（每套有灵魂主线）② **独家 duotone/haze 影像护城河**（区别「干净配图」）③ **证据方法论叙事**（实证）④ **国内开箱即用**（CC0 国内源前置，不依赖 VPN）。

## 怎么用

对你的 Claude Code / Codex 说：「帮我安装这个 Skill：`github.com/dososo/blcaptain-style-skill`」，然后「用 BLCaptain 把这段内容做成小红书图文」。CLI 进阶见 [README](README.md)。
