# 非复制边界

本项目可以研究优秀开源项目和业界最佳实践，但只学习可迁移的方法，不复制任何上游项目的表达结果。

## 可以学习

- 固定版式工作流。
- 种子模板工作流。
- HTML 到 PNG 渲染链路。
- 资产来源记录。
- 验证门禁。
- 主题限制原则。
- 产品文档组织方式。
- 适合 / 不适合场景边界。

## 不得复制

- 代码。
- 模板。
- CSS。
- HTML / JS 实现片段。
- 资产。
- 配色。
- 字体组合。
- 布局 ID。
- 主题名。
- README / SKILL 文案。
- 视觉构图。
- 视觉身份。

## BLCaptain 边界

我们的目标是复现生产纪律，而不是复现别人的视觉身份。

后续所有 Still Paper / Signal Proof 的合同、样张、模板和审美判断，都必须来自本项目自己的视觉语言。
